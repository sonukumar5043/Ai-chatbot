# Ai-chatbot
conversational AI chatbot built using Python, Gradio, and APIs. It uses custom system prompts, chat history, and memory handling to provide human-like responses in a simple interactive interface.
![imag alt](https://github.com/sonukumar5043/Ai-chatbot/blob/main/Screenshot.png)
## Features

- Human-like conversation
- Custom system prompt
- Chat history support
- Friendly chatbot personality
- Simple Gradio UI
- OpenAI/OpenRouter API integration

## Technologies Used

- Python
- Gradio
- OpenAI API
- OpenRouter API
- dotenv

## How to Run

1. Clone the repository

```bash
git clone https://github.com/sonukumar5043/Ai-chatbot/
```
##Install dependencies
pip install -r requirements.txt
